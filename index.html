Navigating the app is very simple. Simply follow the steps below.

  1. Click the dropdown list on the main page.
  2. Find the item you want to learn more about.
  3. Click that item.
  4. Read the text filled in by your choice.
  5. One last thing,... enjoy the app!

To look up certain areas to help in your adventures, follow the steps below.

  1. Click "Find Locations"
  2. Use the dropdown menu to select what you want to find.
  3. Type in your zip code OR select "Find by GPS"
  4. Enjoy using our resources (Google Maps)
